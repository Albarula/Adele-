# adeleproyecto

Description: 
This is a demo Adele's webpage fanpage


funcionalities:
- Api from ticketmaster that shows the next concerts 
- Playlist with the top 5 songs
- Albums information
- Biography information 
- Newsletter subscription 
- Social media links 

We use: 
-Tailwind css
-Vue cli 3
-Vue Router
-APi ticketmaster
